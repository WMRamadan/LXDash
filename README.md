# LXDash

A Dashboard for LXD/LXC

## Requirements

- Python3
- Pip3

## Setup

1. Create and activate virtual environment and install dependencies:
```bash
virtualenv venv
source venv/bin/activate
pip3 install -r requirements.txt
```

2. Run django server:
```bash
cd lxdash/
python3 manage.py runserver
```
